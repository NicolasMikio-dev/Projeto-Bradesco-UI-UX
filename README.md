# Redesign Digital Bradesco - Trabalho A3 💳 🏦

## 📌 Sobre o Projeto

Este projeto propõe uma modernização estratégica da plataforma digital do Bradesco, focada em elevar a experiência do usuário por meio de uma interface intuitiva, dinâmica e amplamente acessível. Desenvolvido como Trabalho A3 para a Universidade Anhembi Morumbi na qual tem perceria com o Bradesco  , o portal foi transformado de um ambiente de consulta tradicional em um hub de gestão financeira interativo. 

*Aviso: Todas as informações e telas exibidas são fictícias e foram criadas exclusivamente para fins acadêmicos, sem qualquer vínculo com os sistemas reais do Banco Bradesco.*

## 🚀 Instruções de Execução

Pré-requisitos:

* Um navegador web moderno (Chrome, Edge, Firefox, etc.)
* Opcional: Extensão "Live Server" no VS Code para visualizar atualizações em tempo real.

Como executar:
1. Clone o repositório: `(https://github.com/NicolasMikio-dev/Projeto-Bradesco-UI-UX.git)`
2. Abra a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador.

## 🛠️ Ferramentas Utilizadas

* *HTML5*: Estruturação semântica do conteúdo.
* *CSS3*: Estilização avançada utilizando variáveis (`:root`), funções matemáticas nativas (`clamp()`) para responsividade e `clip-path` para recortes geométricos.
* *JavaScript*: Lógica de interatividade, consumo de APIs e navegação dinâmica.

## ✨ Funcionalidades

* 📱 *Layout Responsivo e Geometria Dinâmica*: Adaptação automática de fontes e elementos a qualquer tamanho de tela, com recortes visuais complexos criados via código.
* 📋 *Cadastro e Prevenção de Erros*: Formulário intuitivo que fornece feedback automático de sucesso e limita a entrada de caracteres incorretos.
* 💳 *Seção de Cartões Interativa*: Navegação fluida por arraste (drag), efeitos de *hover* indicando elementos clicáveis e revelação progressiva de menus para evitar poluição visual.
* 📍 *Integração com Mapas*: Busca automática da agência Bradesco mais próxima a partir da digitação do CEP pelo usuário.
* ♿ *Acessibilidade Avançada*: Menu dedicado contendo suporte para Web Libras , navegação por comandos de voz e adaptação de cores para daltonismo.

## 💻 Exemplo de Uso

Navegação e interação na plataforma:

> Interação com Cartões:
> Arraste a lista de cartões para os lados para explorar. 
> Passe o mouse sobre o "American Express Gold Card" -> O cartão sofre uma expansão (zoom) e ganha sombra.
> Clique no cartão desejado -> O fundo da página e os botões de ação se adaptam ao contexto do cartão selecionado.
> 
> Busca de Agência:
> Acesse a seção de mapas.
> Digite o CEP "03166-000" no campo de busca.
> O mapa interativo é atualizado e exibe um marcador na agência Bradesco mais próxima na região.
